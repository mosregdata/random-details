# Случайные реквизиты
Случайная выборка реквизитов реальных ИП и компаний: **5000 штук**.

Поля описывающие объект:
1. **ogrn**: ОГРН
2. **inn**: ИНН
3. **okved**: ОКВЭД 2
4. **type**: тип
   - **legal**: компания
   - **individual**: ИП

> Выбираются случайные 2500 шт. индивидуальных предпринимателей и 2500 шт. компаний.
